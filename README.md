### Wibase-service

This is a service that provides and handles the logic for Wibase , this project is writing in nodejs using typescript.

**Setup**

```
npm install
```

**Run server**
---
```
npm run serve
```

**Run build**
---
```
npm run build
```

**Run dev build**
---
```
npm run build-dev
```

**Run application**
---
```
npm start
```
